# Movie Review Sentiment Analysis

An interactive Streamlit web application that predicts the sentiment of a movie review using a trained Recurrent Neural Network (LSTM) model.

## Setup Instructions

Ensure you have **Python 3.9+** installed on your laptop, then run the following commands:

### 1. Set up a Virtual Environment
```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
# venv\Scripts\activate   # On Windows
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Web Application
```bash
streamlit run app.py
```
